## Fortinet Security Fabric

**Today’s networks are more complex than ever, with a wide variety of devices, applications, and services connecting and sending traffic through them.** // Las redes de hoy son más complejas que nunca, con una amplia variedad de dispositivos, aplicaciones y servicios que se conectan y envían tráfico a través de ellos.

**Managing the security of such scenarios using a case-by-case approach can be a challenge, and it is no longer a feasible way to defend your organization from all threats.** // Gestionar la seguridad de tales escenarios utilizando un enfoque de "caso por caso" puede ser un desafío, y ya no es una forma viable de defender a su organización de todas las amenazas.

### Información adicional y análisis de la imagen:

**The image shows three icons representing the pillars of the Security Fabric: visibility (devices), integration (APIs/Code), and automation (gears).** // La imagen muestra tres iconos que representan los pilares del Security Fabric: visibilidad (dispositivos), integración (APIs/Código) y automatización (engranajes).

**The "case-by-case" approach mentioned in the video refers to "Siloed Security," where different security products don't communicate with each other, creating security gaps.** // El enfoque de "caso por caso" mencionado en el video se refiere a la "Seguridad en Silos", donde los diferentes productos de seguridad no se comunican entre sí, creando brechas de seguridad.

**Fortinet Security Fabric is designed to provide broad visibility, integrated detection of advanced threats, and automated response.** // Fortinet Security Fabric está diseñado para proporcionar una visibilidad amplia, detección integrada de amenazas avanzadas y respuesta automatizada.

## Functions of the Fortinet Security Fabric

**The Fortinet Security Fabric is an enterprise architecture that helps manage this complexity by providing a single-pane-of-glass view of the organization's security posture.** // El Fortinet Security Fabric es una arquitectura empresarial que ayuda a gestionar esta complejidad al proporcionar una vista de "panel único" (single-pane-of-glass) de la postura de seguridad de la organización.

**This enables security teams to quickly identify and respond to security threats.** // Esto permite a los equipos de seguridad identificar y responder rápidamente a las amenazas de seguridad.

**The Security Fabric provides integrated, automated, and coordinated security across the organization's entire network infrastructure.** // El Security Fabric proporciona seguridad integrada, automatizada y coordinada en toda la infraestructura de red de la organización.

## Benefits of Using the Fortinet Security Fabric
**These are a few benefits of using the Fortinet Security Fabric. Click each benefit to learn more.** // Estos son algunos beneficios de usar Fortinet Security Fabric. Haga clic en cada beneficio para obtener más información.

### 1. Unified view of the entire network

**A unified view of the entire network from a single console. The network logical and physical topologies are provided.** // Una vista unificada de toda la red desde una sola consola. Se proporcionan las topologías lógicas y físicas de la red. **These topologies show all devices in the fabric, how they are interconnected, as well as their security details and the interfaces used to communicate with other fabric members.** // Estas topologías muestran todos los dispositivos en el fabric, cómo están interconectados, así como sus detalles de seguridad y las interfaces utilizadas para comunicarse con otros miembros del fabric.

### 2. Object synchronization across devices

**Synchronization of various object types across the fabric. This guarantees consistency among all the device members of the fabric.** // Sincronización de varios tipos de objetos a través del fabric. Esto garantiza la coherencia entre todos los dispositivos miembros del fabric.

### 3. Security rating

**Security Rating uses real-time monitoring to analyze your Security Fabric deployment, identify vulnerabilities, and highlight best practices to improve security and performance.** // Security Rating utiliza el monitoreo en tiempo real para analizar su despliegue de Security Fabric, identificar vulnerabilidades y resaltar las mejores prácticas para mejorar la seguridad y el rendimiento. **Security rating checks are triggered automatically when relevant configuration changes are made, and the results provide a detailed view of your network's security posture by evaluating configurations against best practices and compliance standards such as PCI, CIS, and FSBP.** // Las verificaciones de calificación de seguridad se activan automáticamente cuando se realizan cambios de configuración relevantes, y los resultados brindan una vista detallada de la postura de seguridad de su red al evaluar las configuraciones frente a las mejores prácticas y estándares de cumplimiento como PCI, CIS y FSBP.

### 4. Integration

**Integration with many device types. This includes Fortinet devices, as well as devices and platforms from other vendors through an API.** // Integración con muchos tipos de dispositivos. Esto incluye dispositivos Fortinet, así como dispositivos y plataformas de otros proveedores a través de una API.

### 5. Automatic detection of end devices

**New endpoint devices are automatically detected, identified, and added to the topology. Devices with FortiClient installed provide a better integration with the fabric.** // Los nuevos dispositivos finales (endpoints) se detectan, identifican y agregan automáticamente a la topología. Los dispositivos con FortiClient instalado brindan una mejor integración con el fabric.

### 6. Centralized management

**Centralized management of firmware upgrades of all FortiGate, FortiAP, and FortiSwitch devices from the root FortiGate. Upgrades can be done immediately or scheduled and, in the case of FortiGate, the correct upgrade path is followed, if necessary.** // Gestión centralizada de las actualizaciones de firmware de todos los dispositivos FortiGate, FortiAP y FortiSwitch desde el FortiGate raíz. Las actualizaciones pueden realizarse de forma inmediata o programada y, en el caso de FortiGate, se sigue la ruta de actualización correcta, si es necesario.

### 7. Automation

**Automation capabilities: The network is constantly monitored, and automatic actions can be taken when threats are detected without the intervention of administrators.** // Capacidades de automatización: La red se monitorea constantemente y se pueden tomar acciones automáticas cuando se detectan amenazas sin la intervención de los administradores.

## Structure of the Fortinet Security Fabric

**To implement the Fortinet Security Fabric, you need a minimum of two FortiGate firewalls running in NAT mode.** // Para implementar el Fortinet Security Fabric, necesita un mínimo de dos firewalls FortiGate operando en modo NAT.

**One of the FortiGate devices acts as the root firewall of the fabric.** // Uno de los dispositivos FortiGate actúa como el firewall raíz (root) del fabric.

**You must also centralize logging. This requires one FortiAnalyzer or a supported cloud logging solution.** // También debe centralizar el registro (logging). Esto requiere un FortiAnalyzer o una solución de registro en la nube compatible.

**FortiAnalyzer Cloud or FortiGate Cloud can act as the cloud logging solution.** // FortiAnalyzer Cloud o FortiGate Cloud pueden actuar como la solución de registro en la nube.

**Depending on your scenario, and to increase your visibility and control of the network, it is recommended to also add other optional Fortinet devices and even several third-party products.** // Dependiendo de su escenario, y para aumentar su visibilidad y control de la red, se recomienda agregar también otros dispositivos Fortinet opcionales e incluso varios productos de terceros.

**Each one of these devices adds new capabilities to the security team.** // Cada uno de estos dispositivos añade nuevas capacidades al equipo de seguridad.

**For example, FortiManager can simplify the deployment of security policies among all or specific FortiGate firewalls.** // Por ejemplo, FortiManager puede simplificar el despliegue de políticas de seguridad entre todos o algunos firewalls FortiGate específicos.

**The outer circle of icons represents the extended ecosystem, including FortiAP, FortiSwitch, FortiClient EMS, FortiSandbox, FortiMail, FortiWeb, and more.** // El círculo exterior de iconos representa el ecosistema extendido, incluyendo FortiAP, FortiSwitch, FortiClient EMS, FortiSandbox, FortiMail, FortiWeb y otros.

**NAT (Network Address Translation) mode is a requirement for the Root FortiGate because it typically sits at the edge of the network.** // El modo NAT (Traducción de Direcciones de Red) es un requisito para el FortiGate Raíz porque normalmente se ubica en el borde de la red.

## Verification and Topology Visibility

**Once the downstream device is authorized, it becomes a member of the Security Fabric.** // Una vez que el dispositivo downstream es autorizado, se convierte en un miembro del Security Fabric.

**The root FortiGate will then be able to display the security fabric topology.** // El FortiGate raíz podrá entonces mostrar la topología del security fabric.

**You can view the topology in two ways: Logical Topology and Physical Topology.** // Puede ver la topología de dos maneras: Topología Lógica y Topología Física.

**The Logical Topology view shows the network interfaces and the logical connections between the fabric members.** // La vista de Topología Lógica muestra las interfaces de red y las conexiones lógicas entre los miembros del fabric.

**The Physical Topology view shows the actual physical connections between the devices, including FortiSwitches and FortiAPs.** // La vista de Topología Física muestra las conexiones físicas reales entre los dispositivos, incluyendo FortiSwitches y FortiAPs.

### Información adicional y análisis de las imágenes y el video:

**The video demonstrates the final step of authorization where the administrator clicks "Authorize" on the downstream FortiGate (ISFW) from the Root FortiGate's GUI.** // El video demuestra el paso final de autorización donde el administrador hace clic en "Autorizar" en el FortiGate downstream (ISFW) desde la interfaz gráfica del FortiGate Raíz.

**In the "Physical Topology" view, you can hover over any device to see its status, including the number of connected hosts, security risks, and firmware version.** // En la vista de "Topología Física", puede pasar el cursor sobre cualquier dispositivo para ver su estado, incluyendo el número de hosts conectados, riesgos de seguridad y versión de firmware.

**The "Logical Topology" is especially useful for visualizing how traffic flows through different security layers (e.g., from an Internal Segmentation Firewall to the Edge Firewall).** // La "Topología Lógica" es especialmente útil para visualizar cómo fluye el tráfico a través de diferentes capas de seguridad (por ejemplo, desde un Firewall de Segmentación Interna hasta el Firewall de Borde).

**Synchronization Check: After authorization, you should notice that the "Fabric Status" widget on the dashboard turns green, indicating a healthy connection between all members.** // Verificación de sincronización: Después de la autorización, debería notar que el widget "Fabric Status" en el tablero (dashboard) se pone en verde, indicando una conexión saludable entre todos los miembros.

**Telemetry Protocol: The communication between fabric members uses FortiTelemetry (TCP port 8013), which must be allowed on the interfaces connecting the devices.** // Protocolo de Telemetría: La comunicación entre los miembros del fabric utiliza FortiTelemetry (puerto TCP 8013), el cual debe estar permitido en las interfaces que conectan los dispositivos.
