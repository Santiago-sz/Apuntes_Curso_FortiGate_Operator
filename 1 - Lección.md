#### software-Defined wide area network (SD-WAN)
**Fortigates SD-WAN allows you to multiple internet connections efficiently and securely while load-blancing the traffic across all available links.** 
SD-WAN de Fortigates le permite tener múltiples conexiones a Internet de manera eficiente y segura mientras equilibra la carga del tráfico en todos los enlaces disponibles.
	**FTGO SD-WAN improve application performance and reduces cost by routing traffic dynamically. without needing expensive dedicated circuits.**
	FTGO SD-WAN mejora el rendimiento de las aplicaciones y reduce los costos al enrutar el tráfico de forma dinámica, sin necesidad de costosos circuitos dedicados.
	![[Pasted image 20260226203322.png]]
#### **virtual Domains (VDMs)**
**VDOMs allow you to divide a single FortiGate device into multiple virtual firewalls, each with its own polices interfaces, routing tables, and administrators.**
Los VDOM le permiten dividir un solo dispositivo FortiGate en múltiples firewalls virtuales, cada uno con sus propias interfaces de políticas, tablas de enrutamiento y administradores.

[[Nota de que es y como funciona VDOMD]]


![[Pasted image 20260226203727.png]]

**This feature is especially userful for managed service proveders (MSPs) orlarge organizations with different departments or tenants that require isolated enviroments.
by default, traffic from one VDOM cannot reach another VDOM.
However, it is possible to configure routers to enable traffic flow between VDOMs if needed.
in summary, using VDOMs transforms a single FortiGate into a virtual data center for network security.**

Esta función es especialmente útil para proveedores de servicios gestionados (MSP) o grandes organizaciones con diferentes departamentos o inquilinos que requieren entornos aislados.
De forma predeterminada, el tráfico de un VDOM no puede llegar a otro.
Sin embargo, es posible configurar enrutadores para habilitar el flujo de tráfico entre VDOM si es necesario.
En resumen, el uso de VDOM transforma un único FortiGate en un centro de datos virtual para la seguridad de la red.
